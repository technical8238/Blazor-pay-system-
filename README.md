# Blazorsample

A Blazor project scaffolded with the default .NET 10 Blazor template.

## Development

1. Install .NET 10 SDK
2. Restore packages:
   ```bash
dotnet restore
```
3. Run the app:
   ```bash
dotnet run
```

## Notes

- The app uses Blazor WebAssembly / Server components depending on template.
- Static library files under `wwwroot/lib/` are excluded from Git tracking and should be restored via package manager or LibMan if needed.
